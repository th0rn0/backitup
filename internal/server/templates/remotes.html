Add Google Drive remote

Uses a service account — no browser redirect, no OAuth token expiry. The service account authenticates via a JSON key; rclone handles JWT auth internally.

How to create a Google service account ▸
  1. Open Google Cloud Console → APIs & Services → Library and enable the Google Drive API.
  2. Go to IAM & Admin → Service Accounts → Create Service Account. Give it a name and click through.
  3. On the service account's page go to Keys → Add Key → Create new key → JSON. Download the file.
  4. Paste the contents of that JSON file into the field below.
  5. Grant the service account access to a Drive location — either:
    • Shared Drive (recommended): add the service account email as a member of the Shared Drive. Paste the Shared Drive ID in the optional field below.
    • Regular folder: share a folder in your Drive with the service account email address (found in the JSON under "client_email").
Must match the client's Offsite destination value. The built-in dropdown uses gdrive.
Paste the full contents of the downloaded JSON key file.
The ID from the Shared Drive URL. Leave empty if using a shared regular folder instead.
